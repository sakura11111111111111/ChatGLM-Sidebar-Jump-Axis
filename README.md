## 📖 简介 (Introduction)

原生智谱清言在长对话中难以快速定位历史提问。本项目通过 Tampermonkey 脚本，在页面右侧生成一条**具有呼吸感的“光纤轴”**，串联起所有对话节点。

**核心理念：** 拒绝繁杂，视觉纯净，功能至上。

## ✨ 核心功能 (Features)

*   **🧬 光纤轴设计 (Optic Axis)**: 独创的渐变连线视觉，像光纤一样串联起你的思维火花，滚动时永不断裂。
*   **📍 精准锚点**: 自动识别所有提问 (Q1, Q2...)，点击即可平滑滚动直达。
*   **🔒 布局锁死技术**: 采用 Flexbox 智能锁定，无论对话多长，顶部和底部的电梯按钮永远不被遮挡。
*   **💾 智能会话记忆**: 基于 URL 哈希的本地存储，自动记住你在不同会话中“星标”⭐过的重点段落。
*   **💎 宝石典藏版**: 内置紫晶/七彩宝石皮肤，替代枯燥的默认圆点。
*   **📱 响应式折叠**: 窗口变窄时自动收起，仅留一枚精致的悬浮按钮，互不打扰。
